# auth0-multi-tenancy-starter

## Overview

A production-ready, multi-tenant SaaS scaffold built with **Clean Architecture (SOLID)**, **JWT authentication**, **role-based access control**, and **Docker-first deployment**.

**Stack**: .NET 10 (ASP.NET Core) · Vue 3 + Vite · Auth0 Organizations · Docker Compose

## Key Features

✅ **Multi-Tenant Isolation** — Complete organization-level separation  
✅ **Auth0 Integration** — Modern authentication with Organizations support  
✅ **Clean Architecture** — Domain, Application, Infrastructure, and API layers  
✅ **SOLID Principles** — Maintainable, testable, extensible code  
✅ **Docker-Ready** — Single `docker compose up --build` for full stack  
✅ **Production-Grade** — Error handling, input validation, security headers  
✅ **Full RBAC** — Admin and Member roles with permission enforcement
